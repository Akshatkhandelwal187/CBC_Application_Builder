# Vetted Tech Stacks

1. **Next.js Fullstack (TypeScript)**
   - Best for: Server-rendered apps, SEO, unified codebase.
   - Frontend: Next.js App Router, React, Tailwind CSS
   - Backend: Next.js API Routes (Server Actions)
   - Database: PostgreSQL with Prisma ORM

2. **Vite + Express (TypeScript)**
   - Best for: Decoupled client/server, SPAs.
   - Frontend: React + Vite + Tailwind CSS
   - Backend: Node.js Express + tRPC/REST
   - Database: PostgreSQL with Drizzle ORM

3. **Python FastAPI + React**
   - Best for: Data-heavy apps, AI integrations.
   - Frontend: React + Vite
   - Backend: Python FastAPI
   - Database: PostgreSQL with SQLAlchemy

4. **Django Fullstack**
   - Best for: Rapid prototyping, standard CRUD, admin panels.
   - Frontend: Django Templates + HTMX + Tailwind CSS
   - Backend: Python Django
   - Database: PostgreSQL (Django ORM)

**Decision Rubric:**
- If AI or complex data science: Stack 3
- If strict separation of concerns needed: Stack 2
- If extremely rapid standard CRUD: Stack 4
- Default / modern standard fullstack: Stack 1
